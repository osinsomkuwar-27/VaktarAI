// Dashboard Elements
✍️
Message Input Box
Where the leader types the message they want the avatar to say.
Large multiline textarea — min 4 rows
Character counter bottom right (max 500 chars)
Placeholder: "Type your message here..."
Auto-resize as user types
Clear button (×) to reset
🌐
Language Selector
Dropdown to pick the output language for the avatar.
Dropdown with flag emoji + language name
Options: English, Hindi, Tamil, Telugu, Marathi, Bengali, Kannada + more
Default: English
Shows language code next to name (e.g. "Hindi — hi")
🎭
Tone Selector
Lets user pick emotional tone — or leave on Auto for Soham's AI to decide.
Segmented button group or dropdown
Options: Auto · Urgent · Calm · Inspiring · Formal · Empathetic
Default: Auto (AI detects)
Each tone has a small color indicator or emoji
Selected tone highlights in accent color
🚀
Generate Button
The main action button that triggers the entire pipeline.
Large, full-width, accent colored
Label: "Generate Avatar Video"
Loading state: spinner + "Generating..." text
Disabled when text box is empty
Shows estimated time: "~15 seconds"
🎬
Video Player
Where the generated avatar video plays after generation is complete.
Large 16:9 video player
Play / Pause / Mute / Fullscreen controls
Placeholder when no video yet: avatar silhouette illustration
Download button below player
Share button (copy link)
📊
Stats Bar
4 quick stat cards at the top showing usage at a glance.
Messages Sent today
Languages Used
Last Detected Tone
Videos Generated total
🔍
SSML Preview Panel
Shows the raw SSML that Soham's engine returned — for transparency and debugging.
Collapsible panel — hidden by default, expandable
Monospace font code block
Shows detected tone tag at top
Syntax highlighted SSML tags
Copy button to copy raw SSML
🕐
History Panel
List of previously generated videos in this session.
Each row: message preview + language + tone + timestamp
Click a row to replay that video
Shows last 5 generations
Clear history button
⏳
Loading / Progress State
Shows pipeline progress while video is generating.
Step 1: Translating text... ✓
Step 2: Detecting emotion... ✓
Step 3: Generating voice... ✓
Step 4: Rendering avatar video...
Each step ticks off as it completes

// Note to Osin

Build the Dashboard first — that's the working product. The landing page is just marketing.

The single most important component is the Generate button → Video Player flow. Everything else is secondary.

Connect to Shreeja's backend at POST http://localhost:8000/generate-video with body: { "text": "...", "tone_override": null, "target_language": "hi" }

You get back a video_url — just set that as the src of your video player. Done.
